# DATA-STACK

# 开发起步

基于webpack的开发配置环境，可以使用npm管理项目

```
npm i
npm run dev
npm run build
```
推荐使用yarn管理依赖

```
yarn
yarn add package.name
```

# 版本号规则：
- 修订版本号：日常 bugfix 更新。（如果有紧急的 bugfix，则任何时候都可发布）
- 次版本号：带有新特性的向下兼容的版本。
- 主版本号：含有破坏性更新和新特性。
