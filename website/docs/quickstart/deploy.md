# 快速部署

## 根据任务类型需要部署的大数据组件
* 运行数据同步任务 - Flink（standalone/集群）
* 运行Spark SQL任务 - spark（standalone/集群）

## 基础软件安装
* JDK 8+
* MySQL：5.7.33
* Zookeeper：3.5.7
* Redis：5.0.7

## Taiga 依赖的基础组件
* DatasourceX（数据源插件）：[4.3.0](https://github.com/DTStack/DatasourceX/releases/tag/v4.3.0)
