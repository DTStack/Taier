# 创建数据源

## 创建数据源

**点击按钮，进入数据源添加页面**
![add-source](/img/readme/add-source.png)

**配置数据源信息**

![config-source](/img/readme/config-source.png)

:::tip
如果需要开启高可用配置，则需要点击开启kerberos认证按钮，需要上传kerberos证书
该文件是一个zip压缩文件，里面需要包含.keytab文件和krb5.conf文件
:::

填完必要信息后，可以点击测试连通性按钮来检测该数据源是否可用
